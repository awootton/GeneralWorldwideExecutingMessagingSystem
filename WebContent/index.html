Welcome to the temporary home of the GeneralWorldwideExecutingMessagingSystem demos. v2m3

The multichannel chat demo. Aka WootWideChat

There are some instructions in the demo. If you subscribe to the channels that the 3d demos use you can see the messages being transmitted.

The multiuser VR demo. Three.html

W and S will move forward and backwards. A and D move left and right. R and F move up and down (or, they try to - there's gravity).
Hold down the shift key to move faster.
This demo is more interesting (if you are the only person in the space) if you open up two browser windows side by side.

Another VR demo. Three30.html

There is a JavaScript timer that broadcasts position and velocity messages. The main app listens for the messages and updates movement of the 'avatars'. The space subscription is different that the previous demo. More windows result in for 'bots circling. There are mouse controls. You will see that if you have multiple tabs that when you stitch tabs then one 'bot gets jerky. This is normal behavior for Chrome - it slows down hidden tabs.
